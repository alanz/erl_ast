A Rust library for working with Abstract Syntax Trees of Erlang programs

Documentation
-------------

See [RustDoc Documentation](https://docs.rs/erl_ast).

Installation
------------

Add following lines to your `Cargo.toml`:

```toml
[dependencies]
erl_ast = "*"
```
